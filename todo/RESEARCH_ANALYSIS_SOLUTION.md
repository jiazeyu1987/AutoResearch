# 超简化科研分析系统改造方案 (仅修改提示词和UI)

## 项目概述
将现有的舆情分析系统（BettaFish）快速改造为科研分析系统，只修改提示词和UI文字描述，不改动任何核心代码逻辑。

## 核心改造策略

### 1. 零代码改动原则
- 保持所有现有代码逻辑不变
- 复用所有Agent架构和功能
- 仅修改用户界面文字和提示词内容
- 利用现有系统能力处理科研问题

### 2. 最小改动方案

```
ResearchAnalysisSystem/
├── QueryEngine/                 # 保持不变，仅修改提示词
│   ├── prompts/                 # 修改为科研提示词
│   │   └── prompts.py           # 修改为学术问题分析提示词
│   └── ...                      # 其他完全复用
├── MediaEngine/                 # 保持不变，仅修改提示词
│   ├── prompts/                 # 修改为科研提示词
│   │   └── prompts.py           # 修改为科研数据分析提示词
│   └── ...                      # 其他完全复用
├── InsightEngine/               # 保持不变，仅修改提示词
│   ├── prompts/                 # 修改为科研提示词
│   │   └── prompts.py           # 修改为文献分析提示词
│   └── ...                      # 其他完全复用
├── ReportEngine/                # 保持不变，仅修改模板描述
│   ├── report_template/         # 修改现有模板标题和描述
│   └── ...                      # 其他完全复用
├── templates/                   # 修改前端UI文字
│   └── index.html               # 修改界面文字描述
└── app.py                       # 修改界面标题等文字
```

## 具体改造步骤

### 步骤1: 修改提示词 (所有引擎/prompts/prompts.py)
- QueryEngine: 将舆情分析提示词改为科研问题分析提示词
- MediaEngine: 将多媒体分析提示词改为科研数据分析提示词
- InsightEngine: 将舆情洞察提示词改为文献洞察提示词

### 步骤2: 修改前端UI (templates/index.html)
- 将"舆情分析系统"改为"科研分析系统"
- 将"请输入舆情分析需求"改为"请输入科研分析问题"
- 修改示例问题为科研相关示例

### 步骤3: 修改应用标题 (app.py)
- 将Flask应用标题改为"科研分析系统"
- 修改相关界面文字描述

### 步骤4: 修改报告模板描述 (ReportEngine/report_template/)
- 保持模板结构不变
- 修改模板标题和描述文字为科研相关

## 实施内容详情

### 1. 提示词改造示例

#### QueryEngine提示词修改：
```
# 原有舆情提示词：
"请分析关于'{query}'的最新舆情信息，包括主要观点、情绪倾向和关键事件。"

# 修改为科研提示词：
"请检索关于'{query}'的最新学术研究，包括主要研究方向、核心发现和学术观点。"
```

#### MediaEngine提示词修改：
```
# 原有多媒体提示词：
"请分析与'{query}'相关的多媒体内容，包括图片、视频和社交媒体帖子。"

# 修改为科研提示词：
"请分析与'{query}'相关的科研数据和图表信息，包括实验结果和可视化内容。"
```

#### InsightEngine提示词修改：
```
# 原有舆情洞察提示词：
"基于以上信息，请深入分析'{query}'的舆情发展趋势和潜在影响。"

# 修改为科研洞察提示词：
"基于以上信息，请深入分析'{query}'的学术研究趋势和发展前景。"
```

### 2. UI界面文字修改

#### 主界面标题：
- 原标题："微舆 - 多智能体舆情分析系统"
- 新标题："科研智析 - 多智能体科研分析系统"

#### 输入提示：
- 原提示："请输入舆情分析需求，系统将自动进行深度分析..."
- 新提示："请输入科研分析问题，系统将自动进行深度研究..."

#### 示例问题：
- 原示例："分析近期AI技术发展对就业市场的影响"
- 新示例："分析近期大语言模型在医疗诊断中的应用研究"

### 3. 报告模板描述修改

#### 报告标题：
- 原标题："舆情分析报告"
- 新标题："科研分析报告"

#### 报告描述：
- 原描述："基于多源信息的舆情深度分析报告"
- 新描述："基于多源学术信息的科研深度分析报告"
